Six Images

Images - Description

For my web projects, I designed and build two websites. The first one is for a Landscaping company. The website provides information about the companies' activities. Five of the royalty free high quality images were bought from Shutterstock to attract customers' attention and ensure a pleasant look, others were downloaded from a free images database. The Second werbsite is build for a fictional company that offers flower bulbs. All the images related to plants and flowers were taken by Flower Council of Holland website. They could be downloaded for free of charge and royalty-free from their Image Bank, however it is not allowed to change them. In this portfolio are included only images for the Landscaping website.

Images

Image 1 - Edited

Edited Image (JPEG, 317KB, 899 x 600, Produced on Adobe Photoshop 21.2)


Image Manipulation:

  • The picture is resized from its original resolution to 900 x 600px by using Bicubic Sharper (reduction).
  • During the image manipulation are used filters, smart objects and blending modes.
  • A duplicated layer is converted to smart object. A Smart object is simply a shell that protects the layer, and we can apply transformations, adjustments, and filters non-destructively.
  • Another layer called Inversion is created, in order to start creating the main drawing effect. The colours on this layer are inverted.
  • The blending mode is changed to Colour dodge, which makes the image to disappear.
  • Then Filter-Blur-Gaussian Blur… The layer was blurred causing the drawing effect. This is basically a colour pencil drawing effect. High settings of radius: 31 pixels is used to ensure a lot of detail.
  • Pictured is turned to non-destructively black and white adjustment layer. And this adjustment layer turn everything below black and white. Any of the smart filters are still editable by double click on the labels.

Optimisation for Web:

  • JPG is used for web optimisation. In this image there is no extreme changes of colour or hard edges, no need of transparency that can be ensured with PNG, so I used JPG format. Progressive JPG can be used as well. All the manipulations over the JPG images were done at once or in a small number of subsequent saves, since the quality of a JPG can be decreased in multiple subsequent saves.

A subtle approach is used to make colours pop in Adobe Photoshop in a natural way. The main technique is adding saturation and vibrance to make the colours pop.

Original dimensions, size, and type – 1920 x 1279 – 831 KB - JPG

Images

Image 2 Edited

Edited Image (JPEG, 142KB, 900 x 600, Produced on Adobe Photoshop 21.2)


Image Manipulation:

  • First step could be to create a new Hue and Saturation Adjustment Layer to increase the saturation, however another technique is used.
  • Vibrance adjustment layer is used in the beginning. This adjustment layer has also a Saturation layer and a Vibrance Layer. Vibrance and saturation both add saturation to the image.
  • Saturation simply bumps the saturation of all pixels equally. It just increases saturation. Vibrance on the other hand, protects already saturated pixels. So when vibrance is increased, the effect is not as strong as saturation, and it’s more pleasing to the eye.
  • Increasing vibrance is recommended to start with to make colours pop in a photo.
  • If there is still need to make the colours pop even further, a subtle saturation adjustment can be added to make the colours look natural (grass in the image).
  • The specific colour of the grass is targeted to increase the saturation of those colours.
  • This can be done in one of two ways. A Hue and Saturation Adjustment Layer was used for that.
  • After clicking on the icon and hovering over the colour of the grass, Photoshop automatically selects the colour. In this case the grass colour, I selected was found in the yellows. And I could confirm that right below the panel. This is the way to selectively pop the grass colour.

Optimisation for Web:

  • JPG is used for web optimisation, since it is a compressed file format for photographs and realistic images. In this image there is no extreme changes of colour or hard edges, no need of transparency, so I used JPG format. Progressive JPG can be used as well. All the manipulations over the JPG images were done at once or in a small number of subsequent saves in Adobe Photoshop.

Image 4

Description: This image is downloaded from a free images database for my Landscaping project few months ago. However, a link to the site is not kept. This image is reduced and a technique of sharpening is applied.

Original dimensions, size, and type – 1199 x 800, 120 KB, JPG

Images

Image 4 Edited

Edited Image (JPEG, 297KB, 900 x 602, Produced on Adobe Photoshop 21.2)


Image Manipulation:

  • The original layer is duplicated and converted to a Smart object.
  • Image is resized from its original resolution to 900 x 600 px by using Bicubic Sharper (reduction).
  • Then is used Filter-Sharpen-Smart Sharpen… The zoom was on 100%, while working on sharpening. This method gives more control and the algorithm is different than the oldest technique. The amount slider sets the sharpening that is going to be applied on the image. A higher value increase the contrast between the edge pixels, giving the appearance of greater sharpness.
  • The amount of sharpness is set to 72% - a relatively small amount in order to achieve a subtle effect and to avoid edge halos.
  • The radius determines the number of pixels surrounding the edge pixels affected by the sharpening. In this case, I set 1 pixel from the edge to apply the sharpening.
  • Reduce Noise reduces some of the noise found in the image. There may be noise in the sky and other areas that should have any sharpening applied, the noise is reduced to 16%.
  • I chose Gaussian Blur as an algorithm for sharpening.
  • This method also gives option to control Shadows/Highlights from the expandable menu. Fade Amount, Tonal Width and Radius can be controlled on both Shadows and Highlights.
  • Fade Amount allows us to reduce the sharpening applied to the image and the shadows.
  • Tonal Width controls the range of tones and shadows or highlights that are modified. In other words, it controls how dark the shadows are that going to be affected. A lower tonal width will only affect the deep shadows and a larger tonal width will affect the deep shadows and the dark grays. Shadows Fade Amount – 7%, Tonal Width – 9% is chosen.
  • In the Highlights, the Tonal Width, controls the whites and light grays. Highlights Fade Amount – 10%, Tonal Width – 22%, Radius – 1px.
  • 10%, Tonal Width – 22%, Radius – 1px. Camera Raw filter is another method that can be applied for sharpening this image.
